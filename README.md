# Daisy App

## Rubyのバージョン
- ruby 3.3.0
- rails 6.1.7.7

## アプリケーションの実行手順（クローンから起動までの手順）

```sh
git clone gitのURL記入
cd Daisy_app
bundle install
yarn install
rails db:create
rails db:migrate
rails s
```
